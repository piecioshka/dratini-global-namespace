# dratini-global-namespace

:hammer: Return all custom global properties.

## Usage

In HTML file:

```html
<head>
    <!-- As first script! //-->
    <script src="./lib/dratini.js"></script>
</head>
```

**WARNING**

To get best results please add as first script on the page.
